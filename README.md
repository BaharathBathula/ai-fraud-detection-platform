# AI Fraud Detection Platform

A production-style AI fraud detection platform built with FastAPI, Python, and scikit-learn.

## Goals
- Simulate real-world transaction fraud detection
- Train a machine learning model for fraud classification
- Expose fraud scoring through an API
- Track performance metrics for O-1 style evidence and portfolio positioning

## Tech Stack
- Python
- FastAPI
- scikit-learn
- pandas / numpy
- GitHub Codespaces
- Docker

## Planned Features
- Synthetic transaction generation
- Model training pipeline
- Fraud scoring API
- Metrics logging
- Architecture documentation
- O-1 evidence summary

## Relevance
This project supports:
- Original contributions
- Technical authorship
- Public portfolio evidence
